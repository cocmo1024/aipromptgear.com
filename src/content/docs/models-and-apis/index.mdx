## Routing questions

<Steps>
	1. What minimum capability is required for the task to clear the bar?
	2. Which requests justify slower or more expensive reasoning?
	3. What should happen when a provider degrades, rate limits, or changes behavior?
	4. Which tasks need deterministic formatting, policy enforcement, or tool calling support?
</Steps>
