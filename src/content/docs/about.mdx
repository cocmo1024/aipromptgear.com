AI Prompt Gear is an English-language reference site for people building with prompts, agents, and model APIs in real operating environments.

## Scope

- Prompt workflow design and reusable operating patterns
- Model and API selection for practical business constraints
- Tooling, testing, and evaluation for prompt-driven systems

## Audience

This site is built for operators, builders, consultants, and technical teams who need prompt systems that are repeatable, reviewable, and easy to improve over time.

## Editorial approach

The site is structured to favor durable reference material over short-lived commentary. Pages are intended to become reusable knowledge assets that can be updated as models, tooling, and implementation patterns evolve.
